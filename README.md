# 📊 1. Simple Linear Regression

## Description
Simple Linear Regression is used to model the relationship between a single independent variable and a dependent variable by fitting the best straight line through the data.

### Workflow
- Import dataset
- Data preprocessing
- Train-test split
- Train Linear Regression model
- Make predictions
- Visualize regression line
- Evaluate model performance

### Key Concepts
- Linear relationship
- Best-fit line
- Coefficient & Intercept
- Prediction
- R² Score

# 📊 2. Multiple Linear Regression

## Description
Multiple Linear Regression predicts the target variable using two or more independent variables, allowing the model to capture the combined influence of multiple features.

### Workflow
- Load dataset
- Data preprocessing
- Encode categorical variables
- Train-test split
- Train model
- Predict target values
- Evaluate performance

### Key Concepts
- Multiple features
- Feature importance
- Linear relationships
- Multivariable prediction

# 📊 3. Polynomial Regression

## Description
Polynomial Regression extends Linear Regression by transforming input features into polynomial terms, enabling the model to learn non-linear relationships.

### Workflow
- Load dataset
- Generate polynomial features
- Train Linear Regression model
- Predict values
- Visualize polynomial curve
- Compare linear and polynomial models

### Key Concepts
- Non-linear relationships
- Polynomial features
- Curve fitting
- Model complexity

# 📊 4. Support Vector Regression (SVR)

## Description
Support Vector Regression (SVR) predicts continuous values while maintaining the maximum possible margin around the regression line. It performs well on both linear and non-linear datasets.

### Workflow
- Load dataset
- Feature scaling
- Train SVR model
- Predict values
- Visualize predictions
- Evaluate model

### Key Concepts
- Support vectors
- Margin
- Kernel functions
- Feature scaling

### Workflow
- Load dataset
- Train Decision Tree model
- Predict values
- Visualize predictions
- Evaluate performance

- Predict values
